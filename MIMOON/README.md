## ✏️ How to Edit Your Store

### Change Store Name, Tagline, Colors, WhatsApp
1. Open your store → **Admin ⚙️** → **Store Settings 🎨**
2. Edit any field
3. Click **💾 Save All Settings**

### Change WhatsApp Numbers
1. Admin ⚙️ → Store Settings → **💬 WhatsApp Contact**
2. Enter your US number: e.g. `+18049415195`
3. Enter your Nepal number: e.g. `+9779867067872`
4. Save

### Add Products
1. Admin ⚙️ → Products tab
2. Fill in the form and click **✅ Add Product**

### Change Shipping Info
- Open `shipping.html` and edit the text directly

### Change Returns Policy
- Open `returns.html` and edit the text directly

## 🚀 How to Upload to GitHub

1. Go to github.com → **Micollection/Mimoon-store**
2. For each file:
   - Click the file name
   - Click ✏️ pencil icon
   - Select all (Ctrl+A), delete, paste new content
   - Click **Commit changes**
3. Vercel auto-deploys in ~1 minute ✅
